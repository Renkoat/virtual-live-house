フィボナッチ数列とは

フィボナッチ数列は、各項がその前の2つの項の和になっている数列です。通常、最初の2つの項は0と1と定義されます。

**例:** 0, 1, 1, 2, 3, 5, 8, 13, 21, ...

Swiftにおけるフィボナッチ数列の例

`eiπ.ipynb` には、Swift言語でフィボナッチ数列を生成するためのシーケンスの例が記載されています。 

var fib = sequence(state: (0, 1)) {
    (num: inout (Int, Int)) -> Int? in
    let (r, n) = num; num = (n, r + n)
    return r
}

フィボナッチ数列の総和

`eiπ.ipynb` には「フィボナッチ数列の総和は -1」という興味深い記述があります。これは、発散する級数に対する解析接続という数学的な手法で得られる結果です。

その他の関連事項

フィボナッチ数列は、自然界の様々な場所で見られます。例えば、花びらの数、ひまわりの種の配列、松ぼっくりの鱗の配列など、多くの成長パターンに関係しています。
