共役複素数

← Indexに戻る

「共役（きょうやく）」とは、数学において「ペア（対）」をなす特別な関係を意味します。特に複素数の文脈では、「共役複素数」という言葉が、実数軸に対して鏡像の関係にある2つの複素数を指します。

共役複素数の定義

**複素数 $z = a + bi$**（$a, b$ は実数、$i$ は虚数単位）が与えられたとき、その**共役複素数**は、**虚数部分の符号を変えたもの**として定義されます。これは $\bar{z}$（zバー）と書かれます。

もし $z = a + bi$ ならば、その共役複素数 $\bar{z} = a - bi$ です。

  • **例**: 複素数 $3 + 4i$ の共役複素数は $3 - 4i$ です。
  • **例**: 複素数 $5i$（$= 0 + 5i$）の共役複素数は $-5i$ です。
  • **例**: 実数 $7$（$= 7 + 0i$）の共役複素数は、虚数部分が0なので、$7 - 0i = 7$ となり、自分自身になります。

共役複素数の重要な性質

**積と和が実数になる**: 共役複素数と元々の複素数を足したり、掛けたりすると、必ず実数になります。

  • **和**: $(a + bi) + (a - bi) = 2a$ （実数）
  • **積**: $(a + bi)(a - bi) = a^2 + b^2$ （実数）

この性質は、分母に虚数がある分数を有理化する際に使われます。
